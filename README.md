## Project: Employee Data Processing Pipeline

This project simulates a real-world data processing system.

### Problem
Raw employee data often contains missing values, incorrect formats, and invalid records.

### Solution
A Python pipeline processes the raw input by:
- Cleaning invalid records
- Converting data types safely
- Applying business logic
- Generating structured output

### Output Metrics
- Total employees processed
- Total salary
- High salary employee count

This project demonstrates how raw data can be transformed into meaningful insights using structured pipeline design.
